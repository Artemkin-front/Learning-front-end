Сделайте поле ввода(числовое), рядом расположите кнопки, одну для добавления, одну для удаления. При добавлении, число из поля ввода добавляется в массив, при удалении удаляется из него по индексу который ввели в поле ввода. Чтобы демонстрировать результат изменения массива под полем ввода и кнопками расположите блоки которые будут визуализировать массив данных. Под визуализацией массива данных расположите третью кнопку с задачей по варианту:

16 Найти четные повторяющийся элементы(выведите на страницу отдельным блоком)
Добавить чило в массив
Удалить чило из массива по индексу
